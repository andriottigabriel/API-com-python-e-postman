API COM PYTHON + POSTMAN

Este é um código Python que cria uma aplicação de API usando o framework 
Flask para gerenciar informações sobre filmes, permitindo operações básicas,
como consultar, editar, criar e excluir filmes. Essa API pode ser testada usando ferramentas como o Postman 
para enviar requisições HTTP para as diferentes rotas e interagir com os dados de filmes.

API WITH PYTHON + POSTMAN

This is Python code that creates an API application using the framework
Flask to manage movie information, allowing basic operations,
how to consult, edit, create and delete movies. This API can be tested using tools like Postman
to send HTTP requests to the different routes and interact with the movie data.
